# verify — 実装検証ループ

以下の手順で実装を検証する：

1. テストが存在する場合は実行して全件パスを確認
2. サーバー/スクリプトが起動可能か確認
3. 主要な機能が期待通り動作するか確認
4. エラーログ・例外がないか確認

検証結果を「PASS / FAIL + 理由」で簡潔に報告する。FAILの場合は原因の仮説と次の一手を添える。
