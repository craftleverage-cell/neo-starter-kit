# Skill Creator Max

This skill provides a comprehensive, step-by-step guide to creating powerful and effective agent skills. It is based on "The Complete Guide to Building Skills for Claude" and is designed to walk you through the entire skill creation lifecycle, from initial concept to final distribution.

## Core Philosophy

This skill is designed to be an interactive partner in skill development. It will guide you by asking questions one by one, ensuring that all necessary aspects are covered, and referencing a detailed knowledge base for best practices and technical requirements.

## Skill Creation Workflow

The creation process is divided into 6 phases. Follow these steps in order. This skill will guide you through each phase interactively.

**Phase 1: Plan and Design**
- Define the use case, identify the category, and set success criteria.
- *See: `references/phase-by-phase-guide.md`*

**Phase 2: Technical Setup**
- Establish the correct file structure and YAML frontmatter.
- *See: `references/technical-requirements.md` and `references/yaml-frontmatter-guide.md`*

**Phase 3: Write Instructions**
- Draft the main instructions in `SKILL.md` following best practices.
- *See: `references/workflows.md` and `references/output-patterns.md`*

**Phase 4: Develop Resources (Scripts & Templates)**
- Create any necessary scripts, reference documents, or templates.
- *See: `references/mcp-integration-guide.md` for MCP-related development.*

**Phase 5: Test and Iterate**
- Perform comprehensive testing and refine the skill based on results.
- *See: `references/testing-and-iteration.md` and `references/troubleshooting.md`*

**Phase 6: Distribute and Share**
- Prepare the skill for distribution.
- *See: `references/distribution-guide.md`*
