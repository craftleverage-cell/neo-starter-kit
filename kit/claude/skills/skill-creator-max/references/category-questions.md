# Category-Specific Questions

Use these questions to refine your skill based on its category.

## Category 1: Document and Asset Creation

- What is the desired output format (e.g., Markdown, PDF, Python script)?
- Are there any branding guidelines or style guides to follow?
- Is there a template the skill should use for the output?
- What are the key sections or components of the final document/asset?
- How should the skill handle data or input provided by the user?

## Category 2: Workflow Automation

- What are the specific steps in the workflow, in order?
- Are there any decision points or conditional logic?
- What tools (built-in or MCP) are required at each step?
- How should the skill handle errors or failures at each step?
- What is the final, successful outcome of the workflow?

## Category 3: MCP Enhancement

- Which MCP server and tools will this skill use?
- What is the common workflow you want to automate with these tools?
- What domain-specific knowledge or best practices can be encoded?
- How can the skill make the MCP tools easier to use (e.g., by handling context or chaining commands)?
- What are the common mistakes users make with these tools that the skill can prevent?
