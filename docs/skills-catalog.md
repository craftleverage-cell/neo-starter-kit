# スキルカタログ

`kit/claude/skills/` に同梱されている9スキルの一覧。

## このキットの考え方

**このキットのスキルは「仕事の回し方」のみ。** 朝会で全体を見て、計画してから作り、検証して、記録して締める——その運用リズムと品質のフレームワークだけを提供する。

制作系・調査系スキル（スライド生成・Web調査・SNS分析・デプロイ等）は同梱しない。それらは事業や環境ごとに正解が違うため、**各自が `skill-creator-max` で自分の業務に合わせて作る**のがこのキットの設計思想。

---

## 運用リズム系（毎日のサイクル）

| スキル名 | 一言でいうと | 外部依存（CLI/APIキー/アプリ） | 費用注意 |
|---|---|---|---|
| `neo-cockpit` | 朝一・切替時に全プロジェクトの状態（git活動・進捗メモ・インフラ死活）を1画面に集約する司令塔ダッシュボード | git のみ | 特になし |
| `standup` | gitログから今日の作業サマリーを生成 | git のみ | 特になし |
| `wrapup` | セッション終了時に決定事項・次の一手・ブロッカーをメモリへ標準フォーマットで記録 | git のみ（読み取り確認用） | 特になし |

> 推奨リズム: 朝 `neo-cockpit` → 日中は作業 → 夜 `wrapup`。`standup` は日次報告が必要なときに。

## 計画系（作る前に考える）

| スキル名 | 一言でいうと | 外部依存（CLI/APIキー/アプリ） | 費用注意 |
|---|---|---|---|
| `devplan` | 実装前に要件・設計・リスク・工程を構造化し、承認されるまでコードを書かないプランニング関所 | 追加依存なし | 特になし |
| `grilling` | プラン・設計を徹底的に問い詰めて弱点を炙り出すインタビュー | 追加依存なし | 特になし |

## 品質系（出す前に確かめる）

|---|---|---|---|

## 拡張（スキルを増やす）

| スキル名 | 一言でいうと | 外部依存（CLI/APIキー/アプリ） | 費用注意 |
|---|---|---|---|
| `skill-creator-max` | 自分専用スキルの企画〜設計〜テスト〜配布までを対話形式でガイドするスキル作成の教科書 | 追加依存なし（同梱の検証スクリプトを使う場合のみ python3） | 特になし |

## 使い方の補足

- 9スキルすべて、APIキー不要・追加課金なしで動く。外部依存は実質 git のみ（macOS/Linuxなら最初から入っていることが多い）。
- 制作系・調査系のスキルが欲しくなったら、まず `skill-creator-max` を起動して「何をトリガーに・何を出力するスキルか」から設計する。
- 各スキルの詳細な発動条件は各 `SKILL.md` の frontmatter（`description` の Use when / NOT for）を参照。
