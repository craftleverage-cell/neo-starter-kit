# ロードマップ

v1は、環境と「仕事の回し方」のフレームワーク（ルール・エージェント・フックスクリプト・Obsidian連携・スキル9本）に絞って公開している。
制作系・調査系・業務ナレッジ系のスキルを増やす方針は取らず、各自が `skill-creator-max` で育てる設計を軸に据える。

以下はv2以降の検討項目:

- **汎用開発スキル追加の検討** — 事業に依存しない、開発ワークフロー系スキルの追加余地を検討する
- **Windows対応の検討** — 現状はmacOS専用。要望の多さを見て判断する
- **英語版README** — 日本語以外のユーザー向けドキュメントの整備

追加してほしいものの提案は、リポジトリのIssueで受け付けています。
