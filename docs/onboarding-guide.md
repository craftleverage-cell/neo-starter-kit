## Part B — 詳細手順（受け取った人向け）

### 1. 始める前に

始める前に、2つの前提条件を確認してください。どちらか片方でも欠けていると、この先に進めません。

- **Claude Codeの有料プラン**（Pro / Max / Team / Enterprise のいずれか）。無料プランには「Code」機能が含まれていないため、このキットは動きません
- **Mac**（macOS 13 Ventura 以降）。Windowsには対応していません

所要時間の目安は**15〜20分程度**です。途中でターミナルを1回だけ使いますが、そこも含めて誰でもできるように書いてあります。

### 2. Claude Code Desktop アプリを先に用意する

1. https://claude.ai/download を開き、Mac用のインストーラー（.dmgファイル）をダウンロードします
2. ダウンロードした.dmgファイルを開き、中の「Claude」アイコンを「Applications」フォルダにドラッグします
3. 「アプリケーション」フォルダから「Claude」を起動し、Anthropicアカウントでサインインします
4. 有料プランでない場合、画面上部中央の「Code」タブを押すとアップグレードを促す画面が出ます。それが出た場合は先に有料プランへの加入が必要です

このタイミングでは、まだ「Code」タブは使いません。次のステップで、Claude Codeが使う環境（設定・スキル・Obsidian連携など）を先に作ります。

### 3. キットを手に入れる

2つの方法があります。どちらでも構いません。

**方法①: ZIPダウンロード（gitの知識不要、一番簡単）**

1. https://github.com/craftleverage-cell/neo-starter-kit を開く
2. 緑色の「Code」ボタンをクリック
3. 「Download ZIP」をクリック
4. ダウンロードされたZIPファイルをダブルクリックして解凍する（`neo-starter-kit` というフォルダができます）

**方法②: git clone（git操作に慣れている人向け）**

```bash
git clone https://github.com/craftleverage-cell/neo-starter-kit.git
```

### 4. ターミナルを1回だけ使う（ここだけ頑張ってください）

セットアップスクリプトを動かすために、ここだけターミナルを使う必要があります。これは避けられません。以降は基本的にDesktopアプリだけで作業できます。

1. Finderで、解凍した（またはcloneした）`neo-starter-kit` フォルダを開く
2. フォルダを右クリックし、「サービス」→「フォルダに新規ターミナルを開く」を選ぶ
   - この項目が出ない場合は、代わりに次の方法でも同じことができます: `command`＋`スペース`でSpotlightを開き「ターミナル」と入力してターミナルアプリを起動する。開いたウィンドウに `cd ` （半角スペースまで）と入力したあと、Finderの `neo-starter-kit` フォルダをそのままターミナルのウィンドウにドラッグ＆ドロップし、`Enter` を押す

どちらの方法でも、ターミナルが `neo-starter-kit` フォルダの中を指している状態になれば準備完了です。

### 5. セットアップスクリプトを実行

ターミナルに以下を入力して `Enter` を押します。

```bash
./install.sh
```

いくつか質問されるので、以下の通りに答えてください（実際に聞かれる順番で書いています）。

- **「claude コマンドが見つかりません」と出て、インストールするか聞かれた場合** → **「n」と答えてOK**です。Desktopアプリだけを使うなら、ターミナル用のClaude Code（CLI）は不要です
- **Node.jsが見つからない場合** → **「y」でインストールしてもらう**のがおすすめです。Node.jsはObsidian連携（MCPサーバー）に必要で、これはDesktopアプリを使う場合でも必要になります
- **Obsidianアプリが見つからない場合** → **「y」でインストールしてもらう**のがおすすめです
- **jqが見つからない場合** → **「y」**でインストールしてください。危険なコマンドを検知してブロックする安全機能や、監査ログ・Vault自動記録に必要です
- **Vaultの保存場所を聞かれる** → 迷ったら何も入力せずそのまま `Enter` を押せば、初期値（既定のフォルダ）が使われます
- **最終確認** → 内容を確認して「y」で実行します

セットアップが終わると、最後に案内文が表示されます。この中の**「① ターミナルで claude と入力 → ブラウザでログイン」だけは無視してください**。これはターミナル版（CLI）専用の案内で、Desktopアプリだけを使うあなたには不要です。代わりに、このガイドの次のステップ（6と7）に進んでください。

### 6. Obsidianを開く

1. Obsidianアプリを起動する
2. 「保管庫を開く」を選ぶ
3. 先ほどのセットアップで指定したVaultフォルダ（何も入力しなかった場合は既定のフォルダ）を選択する

### 7. Claude Code Desktopで使い始める

1. ステップ2で用意したClaude Code Desktopアプリに戻る
2. 上部中央の「Code」タブをクリックする
3. 環境の選択で「Local」を選ぶ
4. フォルダ選択で、自分の作業用プロジェクトフォルダを指定する（まだ無ければ `~/ClaudeCode` の中に新しくフォルダを作って指定してください）
5. 最初のメッセージとして `/neo-cockpit` と打ってみてください。今日やるべきことの提案が返ってきます

一言だけ注意です。このキットは最初からファイル編集を自動承認する設定になっています（毎回「編集していいですか？」と聞かれません）。慎重に進めたい場合は、Desktopアプリの「Code」タブにあるパーミッションモードの表示から、手動確認モードに切り替えられます。
